# Domena biznesowa

Grupa osób korzystających ze smartfonów, które chcą ustrukturyzować swoje postanowienia i cele, dzieląc się przy tym bieżącymi postępami ze znajomymi.

Obecnie ludzie korzystają z prostych aplikacji, jak np. Notatnik, Reminders itp., a proces dzielenia się progressem jest utrudniony i niewygodny.

Nasze rozwiązanie ułatwiłoby im monitorowanie zadań oraz pozwoliłoby na bieżąco informować znajomych o postępach.
